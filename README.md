# Adivina el número 🎯

## ¿Qué hace?
El programa genera un número aleatorio del 1 al 100 y le pide al usuario que lo adivine. 
En cada intento, el sistema le indica si el número secreto es mayor o menor hasta que lo adivine.

## ¿Cómo ejecutarlo?
1. Clonar el repositorio
2. Abrir el proyecto en NetBeans
3. Ejecutar la clase `EresMayorDeEdad`

## ¿Qué aprendí?
- Uso de `Random` para generar números aleatorios
- Estructuras de control: `do-while`, `if / else if / else`
- Scope de variables en Java
- Subir proyectos a GitHub con Git
